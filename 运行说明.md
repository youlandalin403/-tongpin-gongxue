# 同频共学成熟运行版

这是一个可以直接运行的校园兴趣共学网页作品，重点完成了：

1. 超大兴趣地图：彩色方砖式布局，点击后展示细分方向、推荐小组和行动建议。
2. 共学小组：支持搜索、筛选、查看详情、加入小组和创建小组。
3. 今日打卡：支持选择小组、填写学习主题、时长、状态、心情和学习内容。
4. 资源书院：支持资源筛选、收藏、查看详情和收录资源。
5. 交流庭院：支持发帖、点赞和回复。
6. 成长档案：展示兴趣标签、连续打卡、累计学习时长、收藏资源和小组榜单。
7. 智能学伴：根据方向、状态和可用时间生成今日共学方案。

## 运行方式

方式一：直接打开

进入“网页演示版”文件夹，双击“启动同频共学.html”。

方式二：浏览器打开

进入“网页演示版”文件夹，双击“index.html”。

## 说明

本版本是本地网页演示版，不依赖网络，不需要安装环境。

打开网站后会先进入登录页。可直接使用演示账号：

- 账号：demo
- 密码：123456

也可以注册一个新账号。当前版本会把账号、兴趣方向、加入小组、收藏资源、打卡记录等保存在浏览器本地缓存中，用来模拟真实数据库。

如果要改成真实线上平台，可以把页面里的本地账号、本地数据操作替换成后端接口。你原压缩包中的数据库、服务函数和 C++ 算法模块已经放入“后端与算法模块”文件夹，方便后续联调。
