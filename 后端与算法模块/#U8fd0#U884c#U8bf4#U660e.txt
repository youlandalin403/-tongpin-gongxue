一、项目结构

ConnectHer/
├── cpp_modules/           # C++ 算法模块（不要动）
│   ├── algorithm.exe      # 核心算法程序
│   └── ...
├── pages/                 # 前端页面（美工在这里写）
│   └── （你的 .py 文件）
├── user_service.py        # 用户登录注册
├── group_service.py       # 小组相关
├── checkin_service.py     # 打卡相关
├── post_service.py        # 帖子评论
├── resource_service.py    # 资源收藏
├── auth_service.py        # 权限控制
├── cpp_runner.py          # C++ 调用封装
├── init_db.py             # 初始化数据库
├── init.sql               # 建表语句
├── app.db                 # 数据库文件（运行 init_db.py 生成）
└── main.py                # 入口文件（需要创建）


二、第一次运行（初始化）

1. 打开终端（cmd 或 VS Code 终端）
2. 进入项目目录：cd D:\你的路径\ConnectHer
3. 安装依赖：pip install streamlit
4. 初始化数据库：python init_db.py
   成功后会在当前目录生成 app.db 文件


三、启动项目

方法一（推荐）：
在终端输入：streamlit run main.py

方法二：
如果还没有 main.py，可以直接运行某个页面测试，例如：
streamlit run pages/login.py

五、返回值格式

所有函数返回的都是字典（dict）格式：

成功时：{"success": True, "msg": "...", 其他数据...}
失败时：{"success": False, "msg": "错误原因"}

示例：
result = login("admin", "123456")
if result["success"]:
    uid = result["uid"]
    st.success("登录成功")
else:
    st.error(result["msg"])


六、页面开发示例（登录页）

创建 pages/login.py：

import streamlit as st
from user_service import login

st.set_page_config(page_title="登录")
st.title("同频共学")

username = st.text_input("用户名")
password = st.text_input("密码", type="password")

if st.button("登录"):
    if username and password:
        result = login(username, password)
        if result["success"]:
            st.session_state["uid"] = result["uid"]
            st.success("登录成功")
            st.switch_page("pages/home.py")
        else:
            st.error(result["msg"])
    else:
        st.warning("请填写完整")


七、常见问题

Q1: 提示 No module named 'xxx'
A1: 检查文件名是否正确，比如 user_service.py 不能叫 user.py

Q2: 提示 no such table
A2: 运行 python init_db.py 重新初始化数据库

Q3: C++ 算法调用失败
A3: 确认 cpp_modules/algorithm.exe 存在，且电脑安装了 Visual C++ 运行库

Q4: 数据库 locked
A4: 关闭其他正在访问 app.db 的程序
