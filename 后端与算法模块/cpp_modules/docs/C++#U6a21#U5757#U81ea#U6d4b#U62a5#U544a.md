## 7. 异常情况说明

程序已考虑以下异常情况：

| 异常情况 | 处理方式 |
|---|---|
| 参数数量不足 | 输出 success=false 和错误提示 |
| mode 不存在 | 输出 success=false 和错误提示 |
| input.json 路径错误 | 输出 success=false 和错误提示 |
| 学习时长为负数 | validate 模式输出错误信息 |
| 当前人数超过人数上限 | validate 模式输出错误信息 |
| 用户标签或小组标签为空 | validate 模式输出错误或警告 |
