# 同频共学 C++ 模块调用说明

## 1. 模块定位

本模块是“同频共学”项目中的 C++ 核心计算模块，负责完成以下 4 类计算任务：

| mode | 功能 |
|---|---|
| match | 兴趣匹配算法 |
| score | 打卡积分计算 |
| ranking | 排行榜排序 |
| validate | 数据合法性校验 |

C++ 模块不直接连接 SQLite，不直接处理 Streamlit 页面，只负责读取 JSON、计算结果、输出 JSON。

调用流程：

```text
Python 从 SQLite 查询数据
→ 生成 input.json
→ subprocess 调用 algorithm.exe
→ C++ 读取 input.json 并计算
→ 输出 result.json
→ Python 读取 result.json
→ Streamlit 页面展示结果
```

## 3. 编译方式

### 3.1 使用 Visual Studio Developer Command Prompt

进入 `cpp_modules_submit` 文件夹，运行：

```bat
cl /std:c++17 /EHsc /utf-8 algorithm.cpp /Fe:algorithm.exe
```

或直接运行：

```bat
build_windows.bat
```

### 3.2 使用 MinGW / g++

```bat
g++ algorithm.cpp -std=c++17 -O2 -o algorithm.exe
```

或直接运行：

```bat
build_gcc.bat
```

## 4. 统一调用格式

```bat
algorithm.exe mode input.json result.json
```

参数说明：

| 参数 | 含义 |
|---|---|
| mode | 计算模式，可取 match / score / ranking / validate |
| input.json | 输入 JSON 文件路径 |
| result.json | 输出 JSON 文件路径 |

---

## 5. 四种模式调用命令

### 5.1 兴趣匹配 match

```bat
algorithm.exe match input_samples\match_input_sample.json result.json
```

输出内容包括：

```text
group_id, group_name, match_score, tag_score, active_score, member_score, reason
```

### 5.2 打卡积分 score

```bat
algorithm.exe score input_samples\score_input_sample.json result.json
```

输出内容包括：

```text
valid, checkin_score, base_score, duration_score, streak_score, content_score, content_length, message
```

### 5.3 排行榜 ranking

```bat
algorithm.exe ranking input_samples\ranking_input_sample.json result.json
```

排序规则：

```text
1. score 降序
2. continuous_days 降序
3. total_duration 降序
4. last_checkin_time 升序
```

输出内容包括：

```text
rank, user_id, nickname, score, continuous_days, total_duration, last_checkin_time
```

### 5.4 数据校验 validate

```bat
algorithm.exe validate input_samples\validate_input_sample.json result.json
```

校验内容包括：

```text
学习时长是否为负数
小组人数是否为负数
人数上限是否有效
当前人数是否超过人数上限
标签是否为空
积分、连续打卡天数、学习时长是否为负数
```

## 6. Python subprocess 调用示例

```python
import subprocess

cmd = [
    "cpp_modules/algorithm.exe",
    "score",
    "cpp_modules/input.json",
    "cpp_modules/result.json"
]

result = subprocess.run(cmd, capture_output=True, text=True)

if result.returncode == 0:
    print("C++ module executed successfully")
else:
    print("C++ module failed")
    print(result.stderr)
```

## 7. 注意事项

1. Windows 编译时建议加 `/utf-8`，避免中文编码问题。
2. JSON 字段名统一使用小写字母加下划线，例如 `user_id`、`group_id`、`match_score`。
3. C++ 模块只负责计算，不负责更新数据库。
4. Python 负责生成输入 JSON、调用 exe、读取输出 JSON、更新 SQLite。
5. Streamlit 只负责页面展示。
6. 联调时优先检查路径、字段名、编码和 result.json 是否生成。
